# 🌤️ Dynamic Weather Web Application

A sleek, responsive, and data-driven single-page weather dashboard that queries real-time global climate conditions. Built completely with semantic native technologies and clean structural asset hierarchies.

---

## 🛠️ Tech Stack Used

* **HTML5:** Modular interface structuring (`.card`, `.srch-box`, `.weather`, `.details`).
* **CSS3 Engine:** External typography imports (`Poppins`), custom layout overlays, smooth transition states, and micro-interactions.
* **Vanilla JavaScript:** Asynchronous runtime routines (`async/await`), explicit fetch request handling, and dynamic dynamic view injection nodes.
* **API Framework:** OpenWeatherMap REST API infrastructure endpoints.

---

## ✨ Key Features Included

* **Live Data Sync:** Processes immediate weather attributes (Temperature, Humidity, Wind Speed) based on city parameters.
* **Asynchronous Exception Handling:** Built-in validation structures catch `404` errors to visually toggle invalid search queries immediately (`.error.show`).
* **Conditional Asset Routing:** Dynamic JavaScript nodes replace graphical assets locally depending on target weather metadata conditions (`Clouds`, `Clear`, `Rain`, `Drizzle`, `Mist`).
* **Double-Tiered Media Breakpoints:** Native responsive configurations built for adaptive smartphone, tablet, and widescreen monitor display states.
